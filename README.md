# clicking

clicking is a mini game about _clicking_. its a small test svelte (powered by [`create-svelte`](https://github.com/sveltejs/kit/tree/main/packages/create-svelte)) project.

## developing

```bash
npm run dev
```

## build and preview

```bash
npm run build
npm run preview
```

## deploy

directly to gh pages

```bash
npm run deploy
```
